# 三國萌萌打虛寶序號自動填寫工具

Chrome 插件，用於自動批量填寫三國萌萌打虛寶序號。

## 版本 v1.3.1

- ✅ **更快的反應**：不再等待 3 秒，彈窗出現後立即處理

## 安裝

1. 前往 `chrome://extensions/`
2. 開啟「開發人員模式」
3. 點擊「載入未封裝項目」
4. 選擇此資料夾

## 使用

1. 前往 https://coupon.kingdom-story.com/lang/zh-TW
2. 按 F5 刷新頁面（重要！確保插件已載入）
3. 按 F12 開啟 Console（建議）
4. 點擊插件圖示
5. 上傳主公名稱文件（.txt，每行一個名稱）
6. 上傳虛寶序號文件（.txt，每行一個序號）
7. 選擇執行模式：
   - **一對一**：第1個主公用第1個序號，第2個主公用第2個序號...
   - **共用**：每個序號都給所有主公使用
8. 點擊「開始執行」

## 錯誤處理

插件會自動檢測：
- **console.log 回應**：`{code: 31}` 等錯誤代碼
- **錯誤彈窗**：網站顯示的 "主公名稱輸入錯誤" 等訊息

### 常見錯誤

**code 31** 或 **"主公名稱輸入錯誤"**：
- 主公名稱不存在
- 序號無效或已使用
- 伺服器選擇錯誤（目前固定為「天下爭霸」）

**"Could not establish connection"**：
- 重新載入插件後，必須刷新網頁（按 F5）

## 重要提示

1. **每次重新載入插件後**，必須刷新網頁才能使用
2. **狀態保持**：關閉 popup 再打開時，已上傳的文件和 log 會保留
3. **手動清空**：點擊「清空資料」按鈕可以清空所有儲存的資料
4. 建議開啟 Console 查看詳細執行過程
